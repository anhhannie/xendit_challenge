Question 2: Assume you have to call this endpoint (in microservice A) from another microservice (B). What kind of test case should you prepare to make sure this works accordingly?

We should applied integration testing when microservice(A) call to microservice(B)
  • Verify the response from microservice(B) should be matched with expected result
    • Should included if microservice(B) is under maintainance or down
  • Verify that any new version from microservice(A) should not broken functionality of microservice(B) 
  • And microservice(A) should following any changed from B and update accordingly
  • Verify if request from microservice(A) in queue,  microservice(B) has multiple responses  typical sequential fashion, FIFO (First In, First Out)
